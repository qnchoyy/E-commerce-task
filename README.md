# MyShop - Product Listing Page

A front-end e-commerce Product Listing Page built as a take-home assignment.

## Technologies Used

- ✔️ React + TypeScript (Vite)
- ✔️ CSS Modules
- ✔️ react-hot-toast

## Features

- Sticky header with category navigation (Bags, Shoes, Accessories)
- Product grid with responsive layout (1-4 columns)
- Filter by color and price range
- Sort by name (A-Z, Z-A) and price (Low-High, High-Low)
- Load More
- Add to Cart with toast notifications
- Product counter
- Fully responsive (mobile, tablet, desktop)

## How to Run the Project

1.  Install dependencies
    npm install

2.  Start the development server
    npm run dev
